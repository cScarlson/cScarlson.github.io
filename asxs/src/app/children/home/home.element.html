As we all already know, native Web Components are well-underway to replace traditional magic frameworks. AsXS takes lessons learned from other Web Component frameworks and makes it more native. In fact, your new Microfrontend Framework uses iframes.

You can use AsXS as just a pure CSS SDK with your own stylistic overrides or you can define a Custom Element for it to bootstrap your own behavior. Have a look at the Component Catalog to learn more.
