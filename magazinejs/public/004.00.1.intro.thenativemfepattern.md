
The Native MFE Pattern
================================================================

If you've made it this far, give yourself a big pat on the back for being a gluton for punishment, like me. The last chapter in this book demystifies a couple of misconceptions which, after inventing this pattern, I realized is somewhat of a big deal.

The Native MFE Pattern does not simply advance one language or one capability for specific use-cases; in advances the state of The Web, it turns the entire web infrastucture into a Plugin Architecture in a way that any website or webapp can consume fully-capable components and partials from trusted sites, even if they are simply hosted one's GitHub Pages.
