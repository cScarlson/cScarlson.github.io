
Things you may not have know about in the DOM
================

## Class-List
```javascript
const { classList } = document.querySelector('...');
const active = classList.contains('active');

if (active) classList.remove('active');
else classList.add('active');
```

## Data-Dash-Attributes
```javascript
const { dataset } = document.querySelector('...');
const { x } = dataset;  // access data-x
```

## Attributes are Nodes. Nodes Inherit from EventTarget.
```javascript
const { attributes } = document.querySelector('...');
const { ['class']: klass } = attributes;
const event = new CustomEvent('attr:class', { detail: undefined });

klass.dispatch(event);
```

## Get Form Data
```javascript
const form = document.querySelector('form');
const data = new FormData(form);
const value = data.get('name');
const object = Object.fromEntries(data);
```
